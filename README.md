# 🔮 KatOS
## About
This is the main hub for KatOS, a fast, private, user-friendly operating system. This hub serves as a place to host Stable, Canary, and Nightly builds of KatOS, as well as a means of bug reporting on a system-wide scale. See [the breakdown](#repository-breakdown) for appropriate source code and other components.
